# Infinity-Lang
Infinity Programming Language. A simple weakly-typed, interpreted programming language.

Code Samples comapred to C++:

variable declaration:

'''
int a = 5;
bool b = false;
std::string c = "hello";
'''

'''
let a = 5
let b = false
let c = "hello"
'''

for loop:

'''
for (int i = 0; i < 10; i++)
{
	std::cout << "hello\n";
}
'''

'''
for i = 0 to 10 inc 1 then
	std::println("hello")
end
'''
